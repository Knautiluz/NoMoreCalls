# Projeto feito com IA (Vibe Coding) para eliminar spam
### Filtra apenas chamadas da agenda, e apenas os números permitidos. (funcionalidade que faz falta nativamente em android)

No geral o objetivo do app é suprir a falta da funcionalidade de permitir apenas chamadas de números conhecidos, mas isso cria um problema que ligações importantes possam ser bloqueadas, para tal foi adicionada uma feature de chamadas persistentes, onde o usuário consegue adicionar qual a persistência até a ligação ser permitida.
(Existem alguns apps na loja com tal funcionalidade, mas são pagos e cheios de anúncio, então fiz um pra mim. É isso. :P
